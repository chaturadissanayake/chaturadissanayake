### Philosophy

Effective design is a form of public service. It makes data democratic and complex topics accessible to everyone. My process weighs visual impact and ethical accuracy equally, because a beautiful chart that misleads is still a failure. I work best on messy problems: fragmented data, contested narratives, impossible deadlines.
